  • F1X won Functional Errors/C track of APR-COMP 2024
    F1X, a program repair tool described in our TOSEM 2018 paper "Test-equivalence Analysis for Automatic Patch Generation" won the Functional Errors/C track of APR-COMP 2024.
    2024

    B-Assist is the first deployment of automated program repair focused on user experience, as described in our ICSE SEIP'24 paper "User-Centric Deployment of Automated Program Repair at Bloomberg".
    2023

    2023
